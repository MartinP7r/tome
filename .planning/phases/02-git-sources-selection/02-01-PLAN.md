    Create `crates/tome/src/git.rs` with these public(crate) functions:

    1. **`git_command(repo_dir: &Path, args: &[&str]) -> Result<std::process::Output>`** — private helper. Runs `git` with args, `current_dir(repo_dir)`, and chains `.env_remove("GIT_DIR").env_remove("GIT_WORK_TREE").env_remove("GIT_INDEX_FILE")` per D-11/GIT-06. Returns raw output.

    2. **`git_success(repo_dir: &Path, args: &[&str]) -> Result<()>`** — private helper. Calls `git_command`, bails on non-zero exit with stderr message.

    3. **`git_stdout(repo_dir: &Path, args: &[&str]) -> Result<String>`** — private helper. Calls `git_command`, bails on non-zero, returns trimmed stdout.

    4. **`repo_cache_dir(repos_dir: &Path, url: &str) -> PathBuf`** — per D-01. Computes SHA-256 of URL bytes, formats as lowercase hex, returns `repos_dir.join(hex_hash)`. Uses `sha2::{Digest, Sha256}`.

    5. **`ref_spec_for_config(branch: Option<&str>, tag: Option<&str>, rev: Option<&str>) -> Option<&str>`** — returns the ref to pass to `--branch` on clone or to `git fetch origin <ref>`. For branch/tag, returns the value. For rev, returns None (SHA pinning uses a different fetch flow). For all-None, returns None (track remote HEAD per D-03).

    6. **`clone_repo(url: &str, dest: &Path, branch: Option<&str>, tag: Option<&str>, rev: Option<&str>) -> Result<()>`** — per D-04/GIT-02. Runs `git clone --depth 1` to `dest`. If `ref_spec_for_config` returns Some(ref), adds `--branch <ref>`. For rev-pinned repos (rev is Some but ref_spec is None): clone without --branch, then `git fetch --depth 1 origin <sha> && git reset --hard FETCH_HEAD`. Uses the private git helpers but runs `Command::new("git")` directly for clone (no repo_dir yet) with the same env_remove chain.

    7. **`update_repo(repo_dir: &Path, branch: Option<&str>, tag: Option<&str>, rev: Option<&str>) -> Result<()>`** — per D-04/GIT-03. Determines fetch ref: for branch, fetch that branch; for tag, fetch that tag; for rev, fetch that SHA; for none, fetch HEAD. Runs `git fetch --depth 1 origin <ref>` then `git reset --hard FETCH_HEAD`.

    8. **`read_head_sha(repo_dir: &Path) -> Result<String>`** — per GIT-05. Runs `git rev-parse HEAD`, returns the full 40-char SHA string.

    9. **`effective_path(clone_path: &Path, subdir: Option<&str>) -> PathBuf`** — per D-02. If subdir is Some, returns `clone_path.join(subdir)`. Otherwise returns `clone_path.to_path_buf()`.

    10. **`is_git_available() -> bool`** — probes `git --version` (with env clearing). Returns true if exit code 0. For pre-flight check.

    Add `pub(crate) mod git;` to lib.rs module declarations (between `eject` and `install` alphabetically).

    Use `use sha2::{Digest, Sha256};` and `use std::path::{Path, PathBuf};` and `use anyhow::{Context, Result};`.
