**Cache policy** (per D-04, verbatim):
- Internal `RefCell<Option<Vec<InstalledPlugin>>>` (or equivalent — RESEARCH Q #1 recommends `RefCell` for v0.10's single-threaded sync flow).
- First call to `list_installed`/`current_version`/`available` populates the cache.
- Each `install()` or `update()` returning `Ok` automatically invalidates the cache (`*self.cache.borrow_mut() = None`).
- Public `pub fn refresh(&self) -> Result<()>` allows explicit re-query (forces cache populate by calling the underlying snapshot fetcher).

**`available(plugin_id)` signal source** (per D-02, verbatim):
- Reads the cached `claude plugin list --json` snapshot's `errors` field for the entry.
- If the entry's `errors[]` contains a string matching `"not found in marketplace"`, returns `Ok(false)`.
- Otherwise returns `Ok(true)` (entry exists OR errors[] is absent OR empty).
- **Zero extra subprocess calls.**

**JSON shape** (verified live 2026-05-05 against claude 2.1.128, per RESEARCH "claude CLI JSON Shape"):

`claude plugin list --json` returns a flat JSON array of objects. Per-entry shape:

```json
{
  "id": "axiom@axiom-marketplace",
  "version": "3.3.0",                 // sometimes literal "unknown" string
  "scope": "user",
  "enabled": true,
  "installPath": "/path/to/cache/...",
  "installedAt": "2026-...",
  "lastUpdated": "2026-...",
  "errors": ["Plugin X not found in marketplace Y"]   // OPTIONAL — absent when no errors
}
```

**Heuristic stderr -> InstallFailureKind mapping** (per CONTEXT.md "Claude's Discretion" + RESEARCH §"claude CLI JSON Shape" subprocess invocations table):

| stderr substring | -> kind |
|------------------|---------|
| "not found in marketplace" | NotFound |
| "not found" | NotFound (less specific catch — the install error from `update` path) |
| (default) | Unknown |

Future variants (NetworkError, PermissionDenied) ship empty in the heuristic until empirical evidence surfaces — they exist in the enum so the renderer's grouping is forward-compatible.

       ```rust
       /// Run a `claude` subcommand with stdin closed (per D-01).
       ///
       /// Returns the raw `Output` so callers can inspect stdout AND stderr.
       /// Maps `ErrorKind::NotFound` to a clear error message naming the
       /// binary — this should not happen in practice because the adapter
       /// probes `claude --version` at construction, but it provides a
       /// safety net if the binary disappears between construction and use.
       fn run_claude_subcommand(args: &[&str]) -> Result<std::process::Output> {
           match std::process::Command::new("claude")
               .args(args)
               .stdin(std::process::Stdio::null())
               .output()
           {
               Ok(output) => Ok(output),
               Err(e) if e.kind() == std::io::ErrorKind::NotFound => {
                   Err(anyhow::anyhow!("claude CLI not found on PATH"))
               }
               Err(e) => Err(anyhow::Error::from(e).context(format!(
                   "failed to run `claude {}`",
                   args.join(" ")
               ))),
       ```

       ```rust
       /// Marketplace adapter for the Claude Code plugin ecosystem.
       ///
       /// Per CONTEXT.md D-01..D-04: shells out synchronously to
       /// `claude plugin install/update/list`, holds an internal cache of
       /// the `claude plugin list --json` snapshot to amortize subprocess
       /// cost, and auto-invalidates the cache on successful install/update.
       ///
       /// Per D-09: uses default scope (user) — no `--scope` flag.
       /// Per D-03: no subprocess timeout in v0.10.
       pub struct ClaudeMarketplaceAdapter {
           cache: RefCell<Option<Vec<InstalledPlugin>>>,
       }

       impl ClaudeMarketplaceAdapter {
           /// Construct a ClaudeMarketplaceAdapter, probing `claude --version`
           /// to fail fast if the binary is missing.
           pub fn new() -> Result<Self> {
               if !is_claude_available() {
                   anyhow::bail!(
                       "claude CLI not found on PATH — install Claude Code, \
                        or remove [directories.<name>] entries with \
                        type = \"claude-plugins\" from tome.toml"
                   );
               }
               Ok(Self { cache: RefCell::new(None) })
           }

           /// Test-only constructor that bypasses the binary probe.
           ///
           /// Used by unit tests that exercise trait methods against a
           /// pre-populated cache without requiring `claude` on PATH.
           #[cfg(test)]
           pub(crate) fn new_for_test() -> Self {
               Self { cache: RefCell::new(None) }
           }

           /// Force re-query of the `claude plugin list --json` snapshot.
           ///
           /// Per D-04: callers can use this to refresh the cache mid-flow.
           /// `install()` and `update()` auto-invalidate on success — explicit
           /// `refresh()` is for cases where external state may have changed.
           pub fn refresh(&self) -> Result<()> {
               *self.cache.borrow_mut() = None;
               self.populate_cache()
           }

           /// Run `claude plugin list --json`, parse, and populate the cache.
           ///
           /// No-op if the cache is already populated.
           fn populate_cache(&self) -> Result<()> {
               if self.cache.borrow().is_some() {
                   return Ok(());
               }
               let output = run_claude_subcommand(&["plugin", "list", "--json"])?;
               if !output.status.success() {
                   let stderr = String::from_utf8_lossy(&output.stderr);
                   anyhow::bail!(
                       "`claude plugin list --json` failed: {}",
                       stderr.trim()
                   );
               }
               let stdout = String::from_utf8_lossy(&output.stdout);
               let parsed = parse_claude_plugin_list_json(&stdout)?;
               *self.cache.borrow_mut() = Some(parsed);
               Ok(())
           }

           /// Construct an InstallFailure from a stderr string using the
           /// heuristic classifier.
           pub(crate) fn build_install_failure(
               adapter_id: &str,
               plugin_id: &str,
               op: InstallOp,
               stderr: &str,
           ) -> InstallFailure {
               InstallFailure {
                   adapter_id: adapter_id.to_string(),
                   plugin_id: plugin_id.to_string(),
                   operation: op,
                   kind: classify_claude_install_stderr(stderr),
                   source: anyhow::anyhow!("{}", stderr.trim()),
               }
           }
       }
       ```

       ```rust
       impl MarketplaceAdapter for ClaudeMarketplaceAdapter {
           fn id(&self) -> &str {
               "claude-plugins"
           }

           fn current_version(&self, plugin_id: &str) -> Result<Option<String>> {
               self.populate_cache()?;
               Ok(self
                   .cache
                   .borrow()
                   .as_ref()
                   .and_then(|list| list.iter().find(|p| p.id == plugin_id))
                   .map(|p| p.version.clone()))
           }

           fn install(&self, plugin_id: &str) -> Result<()> {
               let output = run_claude_subcommand(&["plugin", "install", plugin_id])?;
               if !output.status.success() {
                   let stderr = String::from_utf8_lossy(&output.stderr);
                   anyhow::bail!("{}", stderr.trim());
               }
               // Per D-04: auto-invalidate on Ok.
               *self.cache.borrow_mut() = None;
               Ok(())
           }

           fn update(&self, plugin_id: &str) -> Result<()> {
               let output = run_claude_subcommand(&["plugin", "update", plugin_id])?;
               if !output.status.success() {
                   let stderr = String::from_utf8_lossy(&output.stderr);
                   anyhow::bail!("{}", stderr.trim());
               }
               *self.cache.borrow_mut() = None;
               Ok(())
           }

           fn list_installed(&self) -> Result<Vec<InstalledPlugin>> {
               self.populate_cache()?;
               Ok(self.cache.borrow().clone().unwrap_or_default())
           }

           fn available(&self, plugin_id: &str) -> Result<bool> {
               // Per D-02: zero extra subprocess calls. Reads the cached
               // snapshot's errors[] field for the entry.
               self.populate_cache()?;
               let cache = self.cache.borrow();
               let list = cache.as_ref().expect("populate_cache populates the cache");
               match list.iter().find(|p| p.id == plugin_id) {
                   Some(entry) if entry
                       .errors
                       .iter()
                       .any(|e| e.contains("not found in marketplace")) =>
                   {
                       Ok(false)
                   }
                   _ => Ok(true),
               }
           }
       }
       ```
