Add the `similar` crate (3.1.1 MIT, mitsuhiko — package legitimacy audited and approved); add a server-side `tome::machine::preview_save(prefs, path)` helper that returns a `MachineTomlPreview { lines: Vec<DiffLine> }` structured Myers diff between the current and proposed TOML; add two Tauri commands (`preview_machine_toml` produces the structure, `apply_machine_toml` commits via existing atomic `machine::save`); refactor `PreviewPopover` from a hardcoded `dryRunDescription: string` to a content slot (Pitfall 3) — updating the existing Doctor's Fix caller in the same plan to keep the refactor atomic; build `MachineTomlDiff.tsx` as the slot content; wire `[Apply N decisions]` in `TriagePanel` to open the popover.
